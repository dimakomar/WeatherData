## Made with
 * Xcode 9.2 beta (9C34b)
 * Swift 4
 
## Installation

Assuming you have a [Cocoapods](https://cocoapods.org) installed
```bash
 pod install
```
This app is using middleware from [this](https://github.com/dimakomar/weather_data_middleware) repo
